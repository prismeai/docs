## Backup Frequency

| Component | Frequency | Retention |
|-----------|-----------|-----------|
| MongoDB/PostgreSQL | Daily + before updates | 30 days |
| Elasticsearch | Daily snapshots | 14 days |
| Redis | Hourly RDB snapshots | 7 days |
| S3/Object Storage | Versioning enabled | Per policy |
| Kubernetes Config | Before updates | 5 revisions |

## Database Backup

<Tabs>
  <Tab title="MongoDB">
    ```bash
    # Full backup
    mongodump --uri="mongodb://host:27017" --out=/backup/$(date +%Y%m%d)

    # Specific databases
    mongodump --uri="mongodb://host:27017" --db=permissions --out=/backup/permissions
    mongodump --uri="mongodb://host:27017" --db=users --out=/backup/users
    mongodump --uri="mongodb://host:27017" --db=collections --out=/backup/collections
    ```
  </Tab>
  <Tab title="PostgreSQL">
    ```bash
    pg_dump -h host -U user -d permissions > /backup/permissions_$(date +%Y%m%d).sql
    pg_dump -h host -U user -d users > /backup/users_$(date +%Y%m%d).sql
    pg_dump -h host -U user -d collections > /backup/collections_$(date +%Y%m%d).sql
    ```
  </Tab>
  <Tab title="Elasticsearch">
    ```bash
    # Register snapshot repository
    curl -X PUT "http://elasticsearch:9200/_snapshot/backup" \
      -H "Content-Type: application/json" \
      -d '{"type":"s3","settings":{"bucket":"es-backups"}}'

    # Create snapshot
    curl -X PUT "http://elasticsearch:9200/_snapshot/backup/snap_$(date +%Y%m%d)"
    ```
  </Tab>
  <Tab title="Redis">
    ```bash
    # Trigger RDB save
    redis-cli -h redis-host BGSAVE

    # Copy RDB file
    cp /data/dump.rdb /backup/redis_$(date +%Y%m%d).rdb
    ```
  </Tab>
</Tabs>

## Configuration Backup

```bash
# Helm values
helm get values prisme-core -n prisme > /backup/helm-values.yaml

# Kubernetes resources
kubectl get configmaps,secrets -n prisme -o yaml > /backup/k8s-config.yaml

# Workspace configurations (via Builder MCP or CLI)
# Pull all workspaces to local YAML files
```

## Restore Procedures

<Tabs>
  <Tab title="MongoDB">
    ```bash
    mongorestore --uri="mongodb://host:27017" --drop /backup/20260306
    ```
  </Tab>
  <Tab title="PostgreSQL">
    ```bash
    psql -h host -U user -d permissions < /backup/permissions_20260306.sql
    ```
  </Tab>
  <Tab title="Elasticsearch">
    ```bash
    # Close indices first
    curl -X POST "http://elasticsearch:9200/_all/_close"

    # Restore snapshot
    curl -X POST "http://elasticsearch:9200/_snapshot/backup/snap_20260306/_restore"
    ```
  </Tab>
</Tabs>

## Disaster Recovery

| Metric | Target |
|--------|--------|
| RPO (Recovery Point Objective) | < 1 hour |
| RTO (Recovery Time Objective) | < 4 hours |

**DR Testing**: Perform quarterly restoration drills to validate backup integrity.
