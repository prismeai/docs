This page covers environment variables for the core infrastructure services (API Gateway, Runtime, Workspaces, Events). For the platform frontend environment variables, see [Platform Environment Variables](/self-hosting/environment-variables).

## Domains and URLs

| Variable | Service | Description | Default |
|----------|---------|-------------|---------|
| `INTERNAL_API_URL` | All | API Gateway internal URL (ending with `/v2`) | `http://localhost:3001/v2` |
| `API_URL` | All | API Gateway public URL (ending with `/v2`) | `http://studio.local.prisme.ai:3001/v2` |
| `CONSOLE_URL` | api-gateway, pages, runtime | Platform URL for emails, auth redirects | `http://studio.local.prisme.ai:3000` |
| `PAGES_HOST` | api-gateway, pages, runtime | Pages base domain (prefixed with `.`) | `.pages.local.prisme.ai:3100` |

## Redis

| Variable | Service | Description | Default |
|----------|---------|-------------|---------|
| `BROKER_HOST` | All | Redis broker URL (must be same across services) | `redis://localhost:6379/0` |
| `BROKER_PASSWORD` | All | Redis broker password | |
| `BROKER_TLS_CA_FILE` | All | Redis TLS CA filepath | |
| `BROKER_NAMESPACE` | All | Namespace to segment events on shared instances | |
| `BROKER_TOPIC_MAXLEN` | All | Redis streams max length | 10000 |
| `BROKER_EMIT_MAXLEN` | All | Max size (bytes) of emitted events | 100000 |
| `SESSIONS_STORAGE_HOST` | api-gateway | Redis URL for session storage | `redis://localhost:6379/0` |
| `CONTEXTS_CACHE_HOST` | runtime | Redis URL for context persistence | `redis://localhost:6379/0` |

## MongoDB / PostgreSQL

| Variable | Service | Description | Default |
|----------|---------|-------------|---------|
| `PERMISSIONS_STORAGE_DRIVER` | All | Database driver: `mongodb` or `postgresql` | `mongodb` |
| `PERMISSIONS_STORAGE_HOST` | All | Permissions database URL | `mongodb://localhost:27017/permissions` |
| `USERS_STORAGE_DRIVER` | api-gateway | Users database driver | `mongodb` |
| `USERS_STORAGE_HOST` | api-gateway | Users database URL | `mongodb://localhost:27017/users` |
| `COLLECTIONS_STORAGE_DRIVER` | runtime | Collections database driver | `mongodb` |
| `COLLECTIONS_STORAGE_HOST` | runtime | Collections database URL | `mongodb://localhost:27017/collections` |

All database connections support `*_TLS_CA_FILE` and `*_TLS_SELF_SIGNED` options.

## Authentication and Security

| Variable | Service | Description | Default |
|----------|---------|-------------|---------|
| `JWKS_ROTATION_DAYS` | api-gateway | JWT key rotation period (days) | 30 |
| `JWKS_KTY` | api-gateway | JWT key type | RSA |
| `JWKS_ALG` | api-gateway | JWT signing algorithm | RS256 |
| `JWKS_SIZE` | api-gateway | JWT key size (bits) | 2048 |
| `ACCESS_TOKENS_MAX_AGE` | api-gateway | JWT expiration (seconds) | 2592000 |
| `AUTH_PROVIDERS_CONFIG` | api-gateway | Path to SSO config file | `authProviders.config.yml` |
| `SUPER_ADMIN_EMAILS` | api-gateway | Comma-separated super admin emails | |

## Rate Limits

| Variable | Service | Description | Default |
|----------|---------|-------------|---------|
| `RATE_LIMIT_AUTOMATIONS` | runtime | Automations/second | 100 |
| `RATE_LIMIT_EMITS` | runtime | Event emits/second | 30 |
| `RATE_LIMIT_FETCHS` | runtime | HTTP fetches/second | 50 |
| `RATE_LIMIT_REPEATS` | runtime | Repeat iterations/second | 1000 |
| `RATE_LIMIT_DISABLED` | runtime | Disable all rate limits | false |

See [Rate Limits](/api-reference/rate-limits) for the burst variants and per-workspace configuration.

## Performance

| Variable | Service | Description | Default |

| `READINESS_CACHE_TTL_MS` | api-gateway | Readiness check cache duration | 30000 |
