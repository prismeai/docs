## Platform Frontend

Set these on the `services/platform` Docker container:

| Variable | Required | Default | Description |
|----------|----------|---------|-------------|
| `API_URL` | Yes | `https://api.prisme.ai/v2` | Backend API base URL |
| `CONSOLE_URL` | No | `https://console.prisme.ai` | Console URL |
| `OIDC_STUDIO_CLIENT_ID` | Yes | `local-client-id` | OIDC client ID |
| `OIDC_PROVIDER_URL` | No | (derived from API_URL) | OIDC discovery endpoint |
| `PAGES_HOST` | No | `.pages.prisme.ai` | Pages hostname pattern |
| `BUNDLE_BASE_URL` | No | `/builtin-bundles` | Builtin bundles base path |
| `ENABLED_AUTH_PROVIDERS` | No | `""` | JSON array of auth providers |
| `DISABLE_LOCAL_SIGNUP` | No | `false` | Disable local signup |

These are injected at container startup into `window.__ENV__` via `docker-entrypoint.sh`.

## Development Environment

For local development, set these in `services/platform/.env.local`:

| Variable | Description |
|----------|-------------|
| `VITE_API_HOST` | Backend API URL (e.g., `https://api.sandbox.prisme.ai/v2`) |
| `VITE_PRISME_TOKEN` | Access token for SDK (bypasses OIDC in dev) |
| `PRISME_TOKEN` | Token for CLI scripts (import, list) |
| `BUNDLE_BASE_URL` | Bundle path (default: `/builtin-bundles`) |

## Workspace Configuration

Workspace-level configuration is set in `index.yml` under `config.value`. Secrets use the `{{secret.NAME}}` pattern.

### AI Governance (`ai-governance-v2`)

| Config | Description |
|--------|-------------|
| `superAdminToken` | `{{secret.adminAccessToken}}` |
| `platform_admins` | Array of user IDs with root access |
| `test_mode.enabled` | Enable test mode (bypasses auth) |

### Agent Factory (`agent-factory`)

| Config | Description |
|--------|-------------|
| `ai_governance_slug` | AI Governance workspace slug |
| `ai_governance_id` | AI Governance workspace ID |
| `tools_memories_slug` | Memories workspace slug |
| `rateLimits.enabled` | Enable rate limiting |
| `defaults.max_turns` | Default max turns per conversation |
| `defaults.token_budget` | Default token budget |
| `defaults.context_budget` | Default context window budget |

### LLM Gateway (`llm-gateway`)

| Config | Description |
|--------|-------------|
| `providers.openai.api_key` | `{{secret.OPENAI_API_KEY}}` |
| `providers.azure_openai.endpoint` | Azure OpenAI endpoint URL |
| `providers.azure_openai.api_key` | `{{secret.AZURE_OPENAI_KEY}}` |
| `providers.anthropic.api_key` | `{{secret.ANTHROPIC_API_KEY}}` |
| `defaults.chat_model` | Default chat model (e.g., `gpt-4o`) |
| `defaults.embedding_model` | Default embedding model |
| `model_aliases` | Map of friendly names to provider model IDs |
| `exclude_params` | Params to exclude per provider |

### Storage (`storage`)

| Config | Description |
|--------|-------------|
| `default_vector_provider` | `opensearch`, `elasticsearch`, or `mock` |
| `default_embedding_model` | Embedding model for RAG |
| `default_embedding_dimensions` | Vector dimensions (e.g., 1536) |
| `vector_store_index_prefix` | Prefix for vector index names |
| `user_data_expires_after` | Temp data TTL in seconds (default: 1800) |

### AI Insights (`ai-insights-v2`)

| Config | Description |
|--------|-------------|
| `agent_factory.slug` | Agent Factory workspace slug |
| `agent_factory.api_key` | `{{secret.AGENT_FACTORY_API_KEY}}` |
| `tools_memories_slug` | Memories workspace slug |
| `analysis.default_model` | LLM for analysis (default: `gpt-4o-mini`) |
| `analysis.batch_size` | Conversations per analysis batch |

### Agent Evaluations (`agent-evaluations`)

| Config | Description |
|--------|-------------|
| `agent_factory.slug` | Agent Factory workspace slug |
| `agent_factory.api_key` | `{{secret.AGENT_FACTORY_API_KEY}}` |
| `scoring.pass_threshold` | Pass score (default: 70) |
| `retention.runs_per_agent` | Runs to keep per agent (default: 10) |
| `batch.stagger_delay_ms` | Delay between batch runs (default: 2000) |

### Bing Search (`tools-search-bing`)

| Config | Description |
|--------|-------------|
| `tenant_id` | Azure AD tenant ID |
| `client_id` | Azure AD client ID |
| `client_secret` | Azure AD client secret |
| `endpoint` | Azure AI Foundry endpoint |
| `token_cache_ttl` | Token cache duration (default: 3300s) |

### Vector Providers

#### OpenSearch (`vector-opensearch`)

| Config | Description |
|--------|-------------|
| `auth_header` | Basic auth header for OpenSearch |

#### Elasticsearch (`vector-elasticsearch`)

| Config | Description |
|--------|-------------|
| `auth_header` | Basic auth header for Elasticsearch |

#### Mock (`vector-mock`)

| Config | Description |
|--------|-------------|
| `mock_latency_ms` | Simulated latency |
| `mock_error_rate` | Simulated error rate (0-1) |

## Prisme.ai Core Services

For the underlying infrastructure services (API Gateway, Runtime, Workspaces, Events), refer to the [existing environment variables documentation](https://docs.prisme.ai/self-hosting/kubernetes/environment-variables).
