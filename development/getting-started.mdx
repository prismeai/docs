## Prerequisites

- **Node.js** 18+ (LTS recommended)
- **npm** 9+
- **Git**
- Access to a Prisme.ai backend (sandbox or local)

## Clone and Install

```bash
git clone https://github.com/prismeai/prismeai-platform.git
cd prismeai-platform
npm install
```

## Environment Setup

Create `services/platform/.env.local`:

```bash
# Backend API
VITE_API_HOST=https://api.sandbox.prisme.ai/v2

# Auth token (for development — get from DevTools > Cookies > access-token)
VITE_PRISME_TOKEN=your-token-here

# For CLI scripts (import, list)
PRISME_TOKEN=your-token-here

# Bundle path
BUNDLE_BASE_URL=/builtin-bundles
```

<Note>
`VITE_PRISME_TOKEN` bypasses OIDC authentication in dev mode. Get the token from your browser's DevTools after logging into the platform.
</Note>

## Start Development

```bash
# Start the platform dev server (Vite on port 3000)
npm run dev --prefix services/platform
```

The dev server:
- Serves the platform shell on `http://localhost:3000`
- Proxies API calls to the configured backend
- Serves builtin bundles from `dist/builtin-bundles/`
- Hot-reloads shell code changes (but **not** builtin app changes)

## Build Builtin Apps

After modifying builtin app code:

```bash
# Build all builtin apps
npm run build:builtin --prefix services/platform

# Then refresh the browser
```

Builtin apps are compiled by esbuild and output to `dist/builtin-bundles/`. The dev server picks them up automatically.

## Project Structure

```
services/platform/
├── src/                    # Platform shell source
│   ├── App.tsx             # Root routes
│   ├── main.tsx            # Entry point
│   ├── components/         # Shell components (Platform, Auth, Chat)
│   ├── hooks/              # Shell hooks (useAuth, useIAMContext, etc.)
│   ├── lib/                # Core libs (SDK, moduleLoader, sharedModules)
│   ├── socle/              # Shared components/hooks/libs for builtin apps
│   ├── providers/          # React context providers
│   ├── locales/            # i18n translations
│   └── styles/             # Global styles
├── builtin-apps/           # Product frontends
│   ├── secure-chat/
│   ├── agent-factory/
│   ├── ai-governance/
│   ├── ai-knowledge/
│   ├── ai-collection/
│   ├── ai-insights/
│   ├── ai-engage/
│   ├── builder/
│   └── hello-world/        # Template app
├── scripts/                # Build and CLI scripts
├── vite.config.ts          # Vite configuration
├── tailwind.config.ts      # Tailwind configuration
└── Dockerfile              # Production build
```

## Common Commands

| Command | Description |
|---------|-------------|
| `npm run dev` | Start Vite dev server |
| `npm run build` | Full build (shell + builtin apps) |
| `npm run build:builtin` | Build only builtin apps |
| `npm run builtin:init <slug>` | Initialize a new builtin app |
| `npm run builtin:import <slug>` | Associate app with backend workspace |
| `npm run builtin:list` | List available workspaces |
| `npm run builtin:manifest` | Regenerate manifests |
| `npm run lint` | ESLint check |
| `npm run typecheck` | TypeScript type check |
| `npm run test` | Run Vitest |
