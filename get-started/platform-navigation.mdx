The Prisme.ai platform offers a comprehensive set of features and products, organized in an intuitive interface. This guide will help you understand how to navigate the platform effectively and find what you need.

## Home Screen

When you log in to Prisme.ai, you'll land on the home screen, which serves as your central hub for Chat, agents, recent canvas, and product access.

<Frame>
  <img src="/images/get-started-dashboard.png" alt="Prisme.ai home screen" />
</Frame>

The home screen includes:

- **Prompt Composer** - Start a Chat request directly from the home screen
- **Product Navigation** - Access products depending on your role
- **Available Agents** - Open agents your organization has made available
- **Recent Canvas** - Return to recent canvas work
- **What's New** - See highlighted platform updates

## Navigation Structure

Prisme.ai's navigation is organized into several key areas:

### Side Navigation Menu

<Frame>
  <img src="/images/get-started-side-navigation.png" alt="Prisme.ai side navigation menu" />
</Frame>

The side navigation menu includes:

- **Logo** - Click to return to the home screen
- **Use** - Chat and Store experiences
- **Create** - Agent Creator, Knowledges, Collections, Builder, and other creation tools
- **Monitor** - Insights and analytics products
- **Run** - Operational products such as Engage, Governe, and Support
- **User area** - Access your profile, terms, privacy policy, and display mode
