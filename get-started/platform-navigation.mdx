## Navigation Tips and Shortcuts

### Keyboard Shortcuts

Prisme.ai supports several keyboard shortcuts to help you navigate more efficiently:

| Shortcut | Description |
|----------|-------------|
| `Ctrl/Cmd + K` | Open global search |
| `Ctrl/Cmd + /` | Show keyboard shortcut help |
| `Esc` | Close current dialog or panel |
| `Ctrl/Cmd + Home` | Return to dashboard |
| `Ctrl/Cmd + 1-7` | Navigate to products 1-7 |

### Quick Navigation Panel

Press `Ctrl/Cmd + K` at any time to open the quick navigation panel:

<Frame>
  <img src="/images/quick-nav.png" alt="Quick navigation panel" />
</Frame>

This panel allows you to:
- Search across the platform
- Jump directly to specific products or features
- Find recent items
- Access common actions

### Breadcrumbs

Breadcrumbs at the top of each page help you understand where you are in the platform hierarchy and navigate back to parent sections:

<Frame>
  <img src="/images/breadcrumbs.png" alt="Breadcrumb navigation" />
</Frame>

### Mobile Navigation

On smaller screens, the navigation adapts to a more compact format:

<Frame>
  <img src="/images/mobile-nav.png" alt="Mobile navigation" />

Key differences in mobile navigation:
- Main menu collapses into a hamburger menu
