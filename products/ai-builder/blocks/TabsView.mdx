---
title: "TabsView Block"
description: "Display content in tabbed interface."
---

[Demo](https://demo.pages.prisme.ai/en/TabsView)

## Minimal usage

```yaml
# Page/Block
- slug: TabsView
  tabs:
    - text: First Tab
      content:
        blocks:
          - slug: RichText
            content: The first tab content
    - text: Second Tab
      content:
        blocks:
          - slug: RichText
            content: The second tab content
```

`tabs` is an array of panels with a title (`text`) and a `content`:
* `text` is a string or a localized string or an object with a `blocks` attribute containing an array of blocks:
  ```yaml
  # Page/Block
  - slug: TabsView
    tabs:
      - text:
          fr: Premier onglet
          en: First tab
        content:
          blocks:
            - slug: RichText
              content: The first tab content
      - text:
          blocks:
            - slug: RichText
              content: Second tab
        content:
          blocks:
            - slug: RichText
              content: The second tab content
  ```
* `content` is an object with a `blocks` attribute containing blocks:  
  ```yaml
  # Page/Block
  - slug: TabsView
    tabs:
      - text: First tab
        content:
          blocks:
            - slug: RichText
              content: The first tab content
      - text: Second tab
        content:
          blocks:
            - slug: RichText
              content: The second tab content
  ```
