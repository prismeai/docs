AI Collection is Prisme.ai's enterprise-grade tabular data management platform that enables users to create AI agents for analyzing and enriching structured data. It offers a powerful yet intuitive interface for uploading, processing, and querying tabular data using natural language, all within a secure and managed environment.

## Core Capabilities

### Data Ingestion and Management

AI Collection provides multiple options for bringing data into the platform:

<Frame>
  <img src="/images/collection-data-ingestion.png" alt="AI Collection Data Ingestion" />
</Frame>

- Manual upload of CSV, Excel, and other tabular formats
- Connection to AI Builder for scraping web data
- Integration with internal tools (CRM, databases, product analytics)
- Automated data refresh and scheduling
- Version control and history tracking

### AI-Powered Data Enrichment

Transform and enhance your data with AI-powered processing:

<Frame>
  <img src="/images/collection-data-enrichment.png" alt="Data Enrichment in AI Collection" />
</Frame>

- Add new columns based on AI analysis of existing data
- Process data row by row with specialized AI agents
- Apply transformation rules using natural language
- Merge and join datasets with intelligent matching
- Detect anomalies and inconsistencies automatically

### Natural Language Querying

Interact with your data using conversational language:

<Frame>
  <img src="/images/collection-nlq.png" alt="Natural Language Querying in AI Collection" />
</Frame>

#### For Business Users
- Ask questions about your data in plain English
- Get immediate answers without writing SQL
- Visualize results instantly with auto-generated charts
- Save frequently used queries for quick access

#### For Technical Users
- Generate optimized SQL from natural language
- Fine-tune generated queries for complex analysis
- Create parameterized query templates
- Audit and optimize query performance

### Custom Views

Create and share personalized data views using natural language:

<Frame>
  <img src="/images/collection-custom-views.png" alt="Custom Views in AI Collection" />
</Frame>

- Define views using conversational descriptions
- Share views with team members and stakeholders
- Set automated refresh schedules for views
- Export views to various formats (CSV, Excel, PDF)
- Embed views in other applications and dashboards

### Integration with Other Prisme.ai Products

AI Collection works seamlessly with other Prisme.ai products:

<Frame>
  <img src="/images/collection-integrations.png" alt="AI Collection Integrations" />
</Frame>

- **AI Builder** - Create data collection agents that feed into AI Collection
- **AI Knowledge** - Provide data-driven tools for knowledge agents
- **AI SecureChat** - Query data directly within conversations
- **AI Governance** - Apply access controls
- **AI Insights** - Track usage patterns and effectiveness

## Getting Started with AI Collection

<Steps>
  <Step title="Access AI Collection">
    Log in to your Prisme.ai instance and select AI Collection from the main menu.
    
    <Frame>
      <img src="/images/collection-access.png" alt="Accessing AI Collection" />
    </Frame>
  </Step>
  
  <Step title="Import your data">
    Upload a dataset or connect to an existing data source.
    
    <Frame>
      <img src="/images/collection-import.png" alt="Importing Data" />
    </Frame>
    
    You can upload CSV files, connect to databases, or pull data from other systems.
  </Step>
  
  <Step title="Enrich your data">
    Use AI agents to add new columns or transform existing data.
    
    <Frame>
      <img src="/images/collection-enrich.png" alt="Enriching Data" />

  </Step>
</Steps>
