Chat is Prisme.ai's enterprise-grade conversational interface that enables all users to interact with AI in a secure, managed environment. Found under the **Use** menu in the sidebar, Chat has two main entry points:

- **Chat** - Your home for starting conversations, attaching files, and reopening recent Canvas work
- **Agents Space** - Discover, browse, pin, and start conversations with agents available in your organization

Chat offers a familiar chat experience similar to consumer AI tools, but with the security, privacy, and integration capabilities required by organizations.

## Who Uses Chat?

Chat is designed for all users within an organization, from executives to front-line employees. No technical expertise is required to start getting value from the platform.

<Tabs>
  <Tab title="Knowledge Workers">
    Knowledge workers use Chat to:

    - Draft and refine documents, emails, and presentations
    - Research information from internal and external sources
    - Summarize lengthy documents and extract key insights
    - Collaborate on content creation using Canvas
  </Tab>
  <Tab title="Customer-Facing Teams">
    Sales, support, and customer success teams use Chat to:

    - Find product information and support documentation quickly
    - Draft personalized customer communications
    - Analyze customer feedback and identify trends
    - Prepare for customer meetings and presentations
  </Tab>
  <Tab title="Technical Teams">
    Developers, data scientists, and technical staff use Chat to:

    - Generate and debug code
    - Document technical solutions
    - Explore data patterns and generate visualizations
    - Troubleshoot technical issues
  </Tab>
  <Tab title="Leadership">
    Executives and managers use Chat to:

    - Prepare for meetings and presentations
    - Analyze reports and extract key insights
    - Draft communications to teams and stakeholders
    - Stay informed about industry trends and developments
  </Tab>
</Tabs>

### Conversational Interface

Chat provides a familiar interface where users can ask questions, choose an agent, pick an available model, and continue previous conversations.

<Frame>
  ![Chat Conversation](/images/securechat-conversation.png)
</Frame>

- Real-time streaming responses
- Rich text formatting
- Code highlighting and code interpreter support when enabled
- Multi-turn conversations with context retention
- Conversation search, recent history, sharing, and cleanup controls
- Suggested prompts from each agent to help users start with the right context

### Document Processing

Upload and analyze documents directly within conversations:

<Frame>
  ![Chat File Upload Picker with Neutral Example Documents](/images/chat-upload-file-picker.png)
</Frame>

- The **Attach file** button is available in the composer
- Support for common business formats such as PDF, Word, Excel, PowerPoint, and images
- Text extraction and OCR capabilities
- Document summarization and analysis
- Question answering based on document content
- Table, chart, and structured content analysis when supported by the selected model and tools

### Canvas Collaboration

Canvas opens beside the conversation when an agent creates or updates an artifact such as a document, report, code file, dashboard, or generated webpage:

<Frame>
  ![Canvas in Chat](/images/securechat-conversation-active.png)
</Frame>

#### For Non-Technical Users

- Rich content editor with intuitive interface
- Intelligent context menu for generating responses, translations, and text refinement
- Preview and edit modes for generated artifacts
- Export controls for supported artifact types

#### For Technical Users

- Advanced code editor with syntax highlighting
- Specialized context menu for code-related operations
- Real-time preview for web content
- Direct export or download of generated code and files

### Multimodal Capabilities

When using compatible LLMs, Chat can process and analyze visual content:

<Frame>
  ![Chat Multimodal Image Analysis Conversation](/images/chat-multimodal-image-analysis.png)
</Frame>

- Image analysis and description
- Visual question answering
- Chart and diagram interpretation
- Document layout analysis
- Audio, image, and file processing when enabled by the selected model and workspace tools

### Agent Tools

Agents can expose specialized tools that run from the conversation, such as business system lookups, project simulators, web search, or code execution. When a tool is available, Chat shows it in the conversation context and in the **Tools** control near the composer.

- Use the agent's suggested prompts to start with a supported workflow
- Check which tools ran below the agent response
- Open generated reports, dashboards, and files in Canvas
- Change the model when your workspace exposes multiple model options

When a tool requires you to authorize Prisme.ai with an external service (Notion, GitHub, Outlook…), a **Connect** button appears inline in the conversation. See [MCP Connections & OAuth](./mcp-connections) for the full flow and admin setup.

## Getting Started

<Steps>
  <Step title="Access Chat">
    Log in to your Prisme.ai instance. In the sidebar, find the **Use** menu and click **Chat**.

    <Frame>
      ![Chat Home Screen](/images/securechat-home.png)
    </Frame>
  </Step>
  <Step title="Start from the home screen">
    The Chat home screen shows the prompt box, quick prompt categories, available agents, and recent Canvas work. You can:
    
    - **Start typing** in the input box to chat with the default agent
    - **Click an agent card** to start a conversation with a specific agent
    - **Open Recent canvas** to continue work on an existing artifact
    - **Go to Agents Space** to browse all available agents
  </Step>
  <Step title="Upload documents">
    Click the **Attach file** button to analyze specific files.

    <Frame>
      ![Chat File Upload Picker with Neutral Example Documents](/images/chat-upload-file-picker.png)
    </Frame>
    Once uploaded, you can ask questions about the document content.
  </Step>
  <Step title="Use Canvas for content creation">
    During a conversation, Canvas opens automatically when creating documents or code. Canvas provides specialized tools for both text and code creation.

    <Frame>
      ![Chat Canvas Preview](/images/securechat-conversation-active.png)
    </Frame>
  </Step>
</Steps>
