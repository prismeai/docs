## GDPR Compliance

- **Data Processing Agreements** (DPAs) available for all customers
- **Data Protection Impact Assessments** (DPIAs) conducted for high-risk processing
- **72-hour breach notification** commitment
- **Right to erasure** — implemented via [GDPR features](/products/ai-insights/gdpr)
- **Data portability** — workspace export capabilities
- **Privacy by design** — built into the platform architecture

## EU AI Act

Prisme.ai's Responsible AI Framework aligns with EU AI Act requirements:

- **Transparency** — Full audit trails for AI operations
- **Guardrails** — Content filtering, PII detection, hallucination checks
- **Human oversight** — Human-in-the-loop approval workflows
- **Risk assessment** — Agent evaluation and testing tools

## Self-Hosted Compliance

Self-hosted deployments offer additional compliance options:

- **On-premises** — Full data control, no external dependencies
- **Private cloud** — Dedicated infrastructure on your cloud provider
- **Air-gapped** — No internet connectivity required
- **Hybrid** — Selective cloud connectivity

## Audit Capabilities

- **Authentication events** — Login, logout, token creation, SSO
- **Authorization events** — Permission changes, role assignments
- **Data events** — CRUD operations, file access, agent interactions
- **System events** — Configuration changes, deployment updates

Access audit data via the Events API or through [AI Governance — Observability](/products/ai-governance/observability).
