## Data Ownership

All data processed through Prisme.ai remains the property of the customer. Prisme.ai acts as a data processor, not a data owner.

## Data Categories

| Category | Examples | Retention |
|----------|----------|-----------|
| **User Data** | Profiles, preferences, authentication | Until account deletion |
| **Content Data** | Documents, conversations, agent configs | Workspace lifecycle |
| **System Data** | Logs, analytics, events | Configurable retention |

## Multi-Tenancy and Isolation

- **Workspace isolation** — Each workspace has its own data scope
- **Organization isolation** — AI Governance enforces org-level boundaries
- **Database separation** — Configurable per-workspace or shared with logical isolation
- **Network isolation** — Kubernetes network policies for service-level isolation

## Data Residency

Self-hosted deployments provide full control over data residency:
- Choose your cloud region and data center location
- All processing stays within your infrastructure
- No data leaves your environment unless explicitly configured

## LLM Privacy

### External AI Providers

When using external LLM providers (OpenAI, Azure OpenAI, Anthropic):
- Data is sent to the provider's API for processing
- Refer to each provider's data usage policy
- Azure OpenAI and Anthropic offer data processing agreements

### On-Premises AI

For maximum privacy, deploy local LLMs:
- All inference happens on your infrastructure
- No data leaves your network
- See [Local LLMs](/self-hosting/enterprise/local-llm) for setup

### LLM Gateway Controls

The [LLM Gateway](/services/llm-gateway/overview) provides:
- Provider-level routing control
- Usage tracking and audit logging
- Model access control per organization
- [Guardrails](/tools/guardrails) for PII detection, content filtering

## Compliance

- **GDPR** — Data processing agreements, right to erasure, data portability
- **CCPA** — Consumer data rights support
- **AI Act** — Responsible AI framework alignment
- **ISO 27001** — Information security management

See [AI Insights — GDPR](/products/ai-insights/gdpr) for the platform's data deletion and anonymization features.

## Admin Controls

| Level | Controls |
|-------|----------|
| Organization | Data retention policies, provider restrictions, audit settings |
| Workspace | Access control, encryption, data scope isolation |
| User | Consent management, data export, account deletion |
